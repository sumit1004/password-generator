The Secure Password Generator is a simple web application that creates strong and random passwords.
Users can choose the password length and select which characters to include.

Objective

Generate secure passwords easily

Avoid weak and predictable passwords

Practice HTML, CSS, and JavaScript

Technologies Used

HTML – Page structure

CSS – Design and responsiveness

JavaScript – Password logic

Features

Adjustable password length (4–32)

Uppercase, lowercase, numbers, and symbols options

One-click copy to clipboard

Responsive and clean UI

How It Works

Select password length

Choose character options

Click Generate Password

Click Copy to copy the password

Conclusion

This project is a beginner-friendly and useful tool for generating secure passwords and learning JavaScript basics.
